# 🌐 Testes de API com Postman

Projeto prático de QA focado em testes de APIs REST utilizando Postman.

---

# 🎯 Objetivo

Realizar testes funcionais em endpoints REST simulando cenários reais de validação de APIs.

---

# 🧪 Testes realizados

✅ GET de usuário válido  
✅ GET de usuário inexistente  
✅ POST para criação de usuário  
✅ Validação de status code  
✅ Validação de resposta JSON  
✅ Testes positivos e negativos

---

# 🛠️ Ferramentas utilizadas

- Postman
- JSONPlaceholder API
- GitHub
- Markdown

---

# 📚 Conceitos praticados

- APIs REST
- Endpoints
- Métodos HTTP
- Body
- JSON
- Status Code
- Testes manuais de API
